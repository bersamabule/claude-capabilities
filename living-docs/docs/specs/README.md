## Quick Start

### Scan for Specs
```
/spec-scan
```

### Check for Drift
```
/spec-drift
```

### Generate Code from Spec
```
/spec-generate [spec-file]
```

### Validate Specs
```
/spec-validate
```

## Supported Spec Types

| Type | Extensions | Detection |
|------|------------|-----------|
| OpenAPI | `.yaml`, `.yml`, `.json` | `openapi:` or `swagger:` |
| GraphQL | `.graphql`, `.gql` | `type Query` |
| Prisma | `.prisma` | `datasource`, `model` |
| Protocol Buffers | `.proto` | `syntax = "proto` |
| JSON Schema | `.schema.json` | `"$schema"` |
| AsyncAPI | `asyncapi.yaml` | `asyncapi:` |

## Commands Reference

| Command | Description |
|---------|-------------|
| `/spec-scan` | Scan codebase for all spec files |
| `/spec-scan:openapi` | Scan for OpenAPI specs only |
| `/spec-scan:graphql` | Scan for GraphQL schemas only |
| `/spec-drift` | Check for implementation drift |
| `/spec-drift [file]` | Check specific spec for drift |
| `/spec-drift:strict` | Strict mode for CI (fails on drift) |
| `/spec-generate [file]` | Generate code from spec |
| `/spec-generate [file] --types` | Generate types only |
| `/spec-generate [file] --client` | Generate API client |
| `/spec-generate [file] --server` | Generate server stubs |
| `/spec-generate:setup` | Set up code generation tooling |
| `/spec-validate` | Validate all specs |
| `/spec-validate [file]` | Validate specific spec |
