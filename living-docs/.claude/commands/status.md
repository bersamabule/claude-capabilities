# Project Status Check

Review the current state of the project and Living Documentation.

## Instructions

1. Read and summarize CLAUDE.md:
   - Current project status
   - Active work items
   - Known issues
   - Recent changes

2. Check the Chronicle:
   - Find the most recent chronicle entry in `docs/chronicle/`
   - Summarize where the last session left off
   - Note any pending tasks or blockers

3. Review recent ADRs:
   - Check `docs/adr/` for recent decisions
   - Note any that are still "Proposed" status

4. Report to user:
   - Project health summary
   - What was happening last session
   - Recommended next steps
   - Any outstanding questions or blockers

## Output Format

Provide a concise status report:

```
## Project Status: [PROJECT_NAME]

### Current State
[Brief health assessment]

### Last Session
[Date]: [Summary of what happened]
Stopped at: [Where work ended]

### Active Work
- [Task 1]
- [Task 2]

### Blockers
- [Any blocking issues]

### Recommended Next Steps
1. [Priority 1]
2. [Priority 2]
```
