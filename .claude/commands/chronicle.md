# Create Chronicle Entry

Create a new chronicle entry for today's development session.

## Instructions

1. Review the work done in this session by looking at:
   - Files that were modified
   - Conversations and decisions made
   - Any issues encountered

2. Create a new chronicle entry at `docs/chronicle/[TODAY'S DATE]-[brief-description].md`

3. Fill in the template with:
   - **Objectives**: What we set out to do
   - **Summary**: 2-3 sentence overview
   - **Work Completed**: Details of changes made
   - **Decisions Made**: Any architectural or design decisions
   - **Issues Encountered**: Problems and their resolutions
   - **Next Session**: Priority tasks and blockers
   - **Context for Future Sessions**: Critical handoff information

4. Update CLAUDE.md:
   - Update the "Active Work" section
   - Update "Recent Changes" table
   - Update "For Next Session" notes

## Template Location
`docs/chronicle/0000-template.md`

## What to Ask User
- What was the main focus of this session?
- Were there any blockers or issues?
- What should the next session prioritize?
